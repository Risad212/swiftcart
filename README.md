# SwiftCart PHP

**SwiftCart PHP** is a professional MVC e-commerce framework featuring custom routing and checkout logic.

## 🚀 Overview
This project is a custom-built PHP framework designed specifically for e-commerce workflows. It moves away from heavy dependencies to provide a fast, controllable environment for managing products, carts, and user checkouts.

## 🛠 Key Features
* **Custom MVC Router:** A high-performance dispatcher that handles URI mapping and subfolder support.
* **Input Sanitization:** Built-in security logic to clean and validate all user-submitted data.
* **Dynamic Data Extraction:** Uses a clean "Controller-to-View" data extraction system to keep HTML separate from logic.
* **E-commerce Logic:** Optimized backend structure for managing a seamless checkout experience.

## 📁 Project Structure
* **/app**: Core logic, Controllers, and Routing classes.
* **/public**: Entry point (`index.php`) and assets (CSS/JS).
* **/views**: Clean templates for the frontend.
* **/projects/checkout**: Subfolder-ready configuration.
